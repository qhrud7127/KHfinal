# 삼부도서관


 ## 삼부도서관 소개
 
> ###  프로젝트 명
>
> 삼부도서관 홈페이지 구현
>
> ### 개발 목적
>
>>**사용자** 
>> - 고객들의 도서 대출 정보를 한눈에 확인할 수 있고 실제 도서관의 책 정보를 쉽게 검색, 추가, 삭제할 수 있다. 
>>
>> - 고객과의 비대면 업무로 인해 사용자가 도서관리에 더 집중할 수 있다.
>>
>>
>>**고객** 
>> - 도서관 홈페이지를 통해 원격 도서 대출이 가능하고 대출 시 자신의 대출 정보를 확인할 수 있다.
>>
>> - 고객끼리 커뮤니티를 통해 소통이 가능하다.
>>
>> - 도서관 이용 정보를 확인할 수 있다.
>>
>
>
> ### 프로젝트 기대효과
>
>>    많은 도서관 홈페이지에서는 고객이 도서를 검색하여 대출 가능 여부만 확인한 뒤 도서관에 방문하기 때문에 
>>
>>    도서관에 방문하더라도 원하는 책을 빌리지 못하는 경우가 있다. 
>>
>>    삼부도서관 홈페이지에서는 홈페이지에서 바로 도서 대출이 가능하기 때문에 그러한 불편함을 줄일 수 있다.
>>
>>    또한 반응형 웹으로 구현하여 모바일로도 모든 기능을 사용할 수 있기 때문에 고객들의 이용 편의성을 높일 수 있다.
>
>



> ## 개발언어 및 도구
>
>>  OS : Windows10
>>
>> Tool : STS, Brakets
>>
>> Language : JAVA , HTML, CSS, JS, jQuery
>>
>> DB : ORACLE, MyBatis
>>
>> Server : Apache Tomcat 
>>
>> CSS : Bootstrap
>>
